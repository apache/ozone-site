# Integrating Ozone Storage With Other Systems

import DocCardList from '@theme/DocCardList';

A key strength of Apache Ozone is its ability to serve as a unified storage platform accessible via multiple protocols. It functions both as a **Hadoop Compatible File System (HCFS)** through its `ofs://` and `o3fs://` interfaces, and as an **S3-compatible object store** via its S3 Gateway. This dual nature allows seamless integration with a wide range of big data processing frameworks, analytics engines, and data management tools.

This section provides guides for configuring various systems within the data ecosystem to leverage Ozone as their storage backend, taking advantage of its scalability, performance, and protocol flexibility.
