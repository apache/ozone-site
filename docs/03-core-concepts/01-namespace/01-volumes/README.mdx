# Volumes

import useBaseUrl from '@docusaurus/useBaseUrl';
import Link from '@docusaurus/Link'; // Use Docusaurus Link for internal navigation

Volumes are the top-level containers in the Ozone namespace, analogous to user accounts or top-level directories. They are used to group buckets and manage ownership and storage limits.
