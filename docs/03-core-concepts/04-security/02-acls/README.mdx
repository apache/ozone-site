# Authorization With ACLs

import useBaseUrl from '@docusaurus/useBaseUrl';
import Link from '@docusaurus/Link'; // Use Docusaurus Link for internal navigation

Ozone provides Access Control Lists (ACLs) to manage user permissions within the namespace after authentication (e.g., via [Kerberos](../01-kerberos.md)). This allows fine-grained control over who can access specific volumes, buckets, and keys.

Ozone supports two primary ACL systems:
